website review

unit 1 outcome 2
2.1

The tools I used in making this blog were Trello (with the Kanban method) and Visual Studio. I am very happy with VS and find it intuitive to use.
I was intially unsure of Trello but have begun to rely on it more since it does seem to make use of the Kanban method quite effectivly.
If I had to perform this project as part of a group I would have relied on Trello more heavily.

2.2

I have only used Visual Studio for three weeks now but I can already tell that it is a piece of software that I would return to to write my own code in future projects.
As the project has continued I have used Trello increasingly, however I feel that it still lacks a way to separate tasks by their usefullness, a process which I currently achieve offline using an Eisenhower matrix.
It would be useful if I could find a way to incorporate the matrix into Trello.

2.3

There is obviously a lot more that I could do with the website.

Specifically I would liked to have added:
-cool backgrounds to each page.
-a navigation bar that highlights options as you move the mouse across it.
-a contact form that accepts input and responds.
-a website logo for each page.

That being said, I am happy with the fact that I did actually complete a website with five pages and content, and I would enjoy extending it if there was more time.
